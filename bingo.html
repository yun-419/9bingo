☁️ 設定 Firebase 雲端同步

五個人要看到彼此資料，需要共用同一個 Firebase 專案。只需一個人設定一次，之後其他人點「複製分享連結」拿到的網址會自動帶上設定，點一下就自動連線 ☁️
📖 從零建立 Firebase 專案（點展開 · 首次設定需 5 分鐘）
  1. 打開 console.firebase.google.com，用 Google 登入
  2. 點 Add project → 輸入名稱（例如 bingo-2026）→ 關閉 Analytics → Create
  3. 左側選單 Build → Realtime Database → Create Database
  4. 地區選 asia-southeast1 (Singapore) → 選 Start in test mode → Enable
  5. 回首頁，點齒輪 ⚙️ → Project settings
  6. 捲到下方 Your apps → 點 </> 網頁圖示 → 取名 → Register app
  7. 複製跳出的 firebaseConfig = { ... } 貼進上面框框 → 按「儲存並連線」
  8. ✅ 之後你按「複製分享連結」拿到的網址會自帶設定，其他 4 人點連結自動連線
💡 30 天後 test mode 過期 → 到 Realtime Database → Rules，把內容改成 {"rules":{".read":true,".write":true}} → Publish
